# 2048-Inspired Puzzle Game (HTML + SCSS + JavaScript Modules)

## Introduction
This project is a **2048-inspired puzzle game** built using **HTML**, **SCSS**, and **JavaScript modules**.  
It was created as a **challenge project** to test and showcase development skills, especially focusing on **JavaScript logic**, **modular code structure**, and **interactive UI design**.

This task was an exercise to strengthen **JavaScript, DOM manipulation, and modular development skills**.

---

## Getting Started

### Installation Process
1. **Download or clone** this repository to your computer.
2. **Open the folder** in Visual Studio Code (or your preferred code editor).
3. Ensure you have the **Live Server extension** installed in VS Code.
4. Open `index.html` and click **Go Live** in the bottom right corner to launch the project in your browser.
